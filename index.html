WEBSITE

C.O Automatisation est un projet de baccalauréat Sti2d spécialisé dans les systèmes informatiques et numériques.
Ce projet est réalisé dans le but d'automatiser la course d'orientation pour la rendre plus pratique et plus rapide. Les feuilles en papiers, les cartes, les poinçons ont donc été retirés pour laisser place à des sites Internet reliés à une base de données et un système de scannage de balise avec des lecteurs mobiles.
Langages utilisés: Html - Css - JavaScript - Php - MySql.
